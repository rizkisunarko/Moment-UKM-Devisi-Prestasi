# Moment UKM Divisi Prestasi

![Repo Stars](https://img.shields.io/github/stars/rizkisunarko/Moment-UKM-Devisi-Prestasi?style=social) ![Forks](https://img.shields.io/github/forks/rizkisunarko/Moment-UKM-Devisi-Prestasi?style=social)

Selamat datang di repositori **Moment UKM Divisi Prestasi**! Repositori ini didesain untuk menyimpan dokumentasi, informasi, dan rekap perjalanan satu periode penuh dari divisi prestasi UKM PSHT Universitas Trunojoyo Madura (UTM).

## Tentang Proyek
Proyek ini dibangun menggunakan kerangka kerja [Bootstrap](https://getbootstrap.com/), salah satu framework CSS paling populer untuk membuat desain website yang responsif dan modern.

Tujuan utama repositori ini:
- Dokumentasi kegiatan divisi prestasi dalam satu periode.
- Penyimpanan catatan capaian dan pencapaian divisi.
- Sebagai referensi dan bahan evaluasi untuk periode berikutnya.

## Teknologi yang Digunakan
Repo ini dibangun dengan menggunakan teknologi berikut:

- **CSS**: Membentuk layout dan desain dengan gaya yang responsif. (~47.4% dari kode repo).
- **HTML**: Struktur dasar halaman website. (~35.9% dari kode repo).
- **JavaScript**: Untuk menambahkan interaktivitas. (~16.7% dari kode repo).

## Cara Mengakses
Kamu bisa mengakses proyek ini dengan cara mengunduh atau mengkloning repositori ini:

### Langkah 1: Clone repositori
```bash
    git clone https://github.com/rizkisunarko/Moment-UKM-Devisi-Prestasi.git
```
### Langkah 2: Buka file pada browser
Buka file index.html dengan menggunakan browsermu untuk melihat proyek ini secara langsung.
